# Relations
	- when two objects are related with each other in database we create relation between them.

# type of relation
	- One to Many : when one object can have multiple relation with related table objects.
	- One to One : When one object have one relation with related btable objects
	- Many to Many : when a-table object can have multiple relation with related b-table object and the related b-table object can have multiple relation 			 with a-table objects
